## Purpose of Each File
- README.md – Program overview, benefits, eligibility, onboarding process, and links to other documents
- PROJECTS.md – Current active projects and available opportunities
- TRAINING.md – Detailed training curriculum and learning roadmap
- CONTRIBUTING.md – GitHub workflow, pull request process, code review expectations, and contribution guidelines
- ROADMAP.md – Future goals and planned initiatives
- CODE_OF_CONDUCT.md – Professional behavior and community expectations

---

## Student Journey

### Month 1 – Foundation
- Introduction to Software Engineering
- Programming Fundamentals
- Git Basics
- GitHub Basics
- Linux Fundamentals
- Documentation Writing

Exercises:
- Create GitHub account
- Fork repositories
- Create pull requests
- Resolve merge conflicts

---

### Month 2 – Engineering Practices
- SDLC
- Agile Methodology
- Requirement Engineering
- JIRA
- Cybersecurity Fundamentals
- Code Review Process

Exercises:
- Review pull requests
- Create user stories
- Create Jira tasks
- Prepare technical documentation

---

### Month 3+ – Project Contribution
Students join one of:
- Research Support Team
- FaaS Platform Team
- Community Infrastructure Team
- AI Agent Team
- SDLC Automation Team

Expected Activities:
- Weekly sprint participation
- Task implementation
- Documentation updates
- Code reviews
- Team collaboration

---

## Evaluation Criteria
- Attendance & Participation – 10%
- Training Completion – 20%
- Technical Assignments – 20%
- GitHub Contributions – 20%
- Project Contributions – 20%
- Documentation & Communication – 10%

---

## Completion Requirements
Students will be considered successful contributors if they:
- Complete mandatory training
- Participate in project activities
- Submit approved pull requests
- Follow review feedback
- Maintain professional conduct
- Complete assigned deliverables

---

## Recognition
Eligible participants may receive:
- Contribution Certificate
- Project Experience Certificate
- Open Source Contribution Record
- Technical Recommendation (based on performance)
- Mentorship Support for Career Development

---

## Contributor Levels Badge System
- 🌱 Trainee Engineer
- 🔧 Associate Contributor
- 🚀 Project Contributor
- ⭐ Senior Contributor
- 🏆 Technical Lead Intern

This badge system helps contributors visualize their progression path and stay motivated as they advance.
