# Requirement Engineering

---

## What is Requirement Engineering?
- The process of identifying, documenting, validating, and managing requirements

---

## Types

### Functional Requirements
- What the system should do  
- Example:  
  - User can deploy an AI agent

### Non-Functional Requirements
- Quality attributes  
- Examples:  
  - Performance  
  - Security  
  - Availability

---

## Requirement Lifecycle
- Gather  
  ↓  
- Analyze  
  ↓  
- Document  
  ↓  
- Validate  
  ↓  
- Implement  
  ↓  
- Maintain
