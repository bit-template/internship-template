# Software Development Life Cycle (SDLC)

## Phases

### Requirements

Understand business needs.

### Design

Create architecture and design.

### Development

Implement the solution.

### Testing

Verify functionality and quality.

### Deployment

Release to users.

### Maintenance

Fix issues and improve features.
