# Cybersecurity Fundamentals

---

## Objectives
- Understand basic security practices

---

## Never Share
- Passwords
- API Keys
- Tokens
- Certificates

---

## Secure Development
- Validate inputs
- Use authentication
- Follow least privilege principle

---

## Common Risks
- Phishing
- Weak Passwords
- Credential Leakage
- Unpatched Software

---

## Reporting
- Report security concerns immediately to mentors
