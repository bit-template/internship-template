Projects related to AI agents, automation services, and intelligent workflows.

### SDLC Automation
Projects that improve software development processes through automation and tooling.

### Community Infrastructure
Projects that enable community-driven resource sharing and collaborative computing.

## Contributor Assignment Process
1. Complete onboarding and training.
2. Select preferred project category.
3. Discuss project allocation with mentor.
4. Receive starter issues.
5. Begin contributing through GitHub workflow.

## Project Status Definitions

| Status     | Meaning                               |
|------------|---------------------------------------|
| Planned    | Idea under evaluation                 |
| Active     | Accepting contributors                |
| Maintenance| Bug fixes and improvements only       |
| On Hold    | Temporarily paused                    |
| Completed  | Project objectives achieved           |

## Proposing New Projects
Contributors may propose new projects by creating a proposal issue containing:
- Problem Statement
- Proposed Solution
- Expected Benefits
- Technical Requirements
- Estimated Effort
- Mentor Availability

All proposals will be reviewed before acceptance.
